# The Forge AFK Miner Bot / Mini Game Forge Helper

This bot is designed to help you AFK mine in The Forge map and includes a system to assist with the State 3 Forge mini-game (accuracy is not yet 100%).

# Getting Started

## Prerequisites

### System

- Python 3.14

- uv

### Game Pass

- Sell Anywhere

## Installing

Sync Use Package

`uv sync`

## Running

`uv run main.py`
